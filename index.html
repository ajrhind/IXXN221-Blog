My website is going well. I forgot to make the repository for it when I started, so it starts with hal;f he website already there. I feel like I'm doing quite well with the progress, I only have a few things left to do.
I have been going through trying to make all my pages look as close to the figma files as I can. I feel like I'm beging quite extra with this thouhg. I really feel like I should be doing all the pages I can, but I know I only have to make it look like a functioning website.

One thing I'm having problems with is that I started designing the website in desktop mode, so I had to change everything to make it responsive for mobile and tablet size screens. I had to go back and watch the repsonsive HTML video multiple times, and I learnt how to do media queries. What they do is change the way something will react if the screen is below a certain size. I've used them to change around the grids when the screen gets to a tablet or phone size.

I've done some cool little animation things with the image zoom when you hover on it, and im hoping to do a few more if I have time. I'm trying to keep the overall design of the website to very minimalistic, and easy to follow. I don't want to makie things too complicated for the viewer.

9/10


With tomorrow being hand in day I feel like I have done all I can for my website. I would have really loved to be able to add in some scrolling animations, and some fading in or out animations, but I didn't have a lot of time.
I am really happy with how it did turn out. I liked the zoom in when you hover over the pictures, and I feel like I executed that aspect of interactive quite well. I also really like the way I designed it, as it feels very minimalistic, and straight foward with the flow. The green feels calming, inviting, fresh, and healthy. All things that I would want to see on a website for learning to cook. I also chose that green specifically because it is cool toned, and that helps with being easy on the eyes, and it is easier for colourblind people to see.

I used this colourblind diagram to help chose a colour for the website.
With this diagram, I focused on the green. It isn't an exact colour match, but I had to try and fit it into somewhere. If we follow this green through, we can see that with the different types of colourblind vision, it is viewed as a tan kind of yellow and a blue. I figured that these would be a good aesthetic as well. I would have preferred to use the purple, but I decided that it doesn't fit at all with what I'm trying to sell.
